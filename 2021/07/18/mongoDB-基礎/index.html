註冊

如有私有伺服器可參考官網架設教學，這邊使用官方免費的 mongoDB Atlas 作範例

  1. 註冊 mongoDB Atlas
  2. 選擇 Shared 免費資料庫方案
  3. 選擇 資料庫位置，這邊以 Google Cloud 在台灣之伺服器作為示範
    選擇完點選建立

    完成註冊後等待約 3 分鐘建立 server 即可開始使用
  4. 取得連線連接
    1. 點選 CONNECT
    2. 設定連接資訊
      因環境 IP 不固定，測試階段可先開啟所有 IP
      選擇 Allow Access from Anywhere 接受所有連線
      底下的 Username 以及 Password 自行設定後點擊 Create Database User 並點擊 Choose a connection method 按鈕
    3. 選擇 Connect your application
    4. 選擇 Python 3.6 就可以取得 Connection String 囉~

條件搜尋符號解釋表

符號 意義 範例
$lt 小於 {'age': {'$lt': 20}}
$gt 大於 {'age': {'$gt': 20}}
$lte 小於等於 {'age': {'$lte': 20}}
$gte 大於等於 {'age': {'$gte': 20}}
$ne 不等於 {'age': {'$ne': 20}}
$in 在範圍內 {'age': {'$in': [20, 23]}}
$nin 不在範圍內 {'age': {'$nin': [20, 23]}}
符號 意義 範例 範例解釋
$regex 搜尋正則表達式 {'name': {'$regex': '^M.*'}} name 以 M 開頭
$exists 屬性是否存在 {'name': {'$exists': True}} name 屬性存在
$type 型態判斷 {'age': {'$type': 'int'}} age 的型態為 int
$mod 數學運算 {'age': {'$mod': [5, 0]}} 年齡除 5 於 0
$text 文本查詢 {'$text': {'$search': 'Mike'}} 查詢所有資料型態為 text 的屬性包含 Mike
$where 高級查詢 {'$where': 'obj.fans_count == obj.follows_count'} 自身粉絲數等于關注數量
